WeatherInfo
===================================

# 功能
一个基于OpenWeather API的天气预报软件，可以查询近7日的天气信息，还可以收藏常用的地点以便随时查看。

# 使用
此应用使用了 [Open Weather](https://openweathermap.org/api) API。如要使用此应用，应先到OpenWeather网站创建密钥，并将密钥按照如下格式 `WeatherApiKey="xxxxxxxxxx"` 填写到local.properties文件。
